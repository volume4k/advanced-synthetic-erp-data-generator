# Repo-Local Agent Assets

This directory contains repository-local guidance that can be used by any AI agent working in this checkout.

## Skills

Skills live under `.agents/skills/`. Agents should read a skill's `SKILL.md` before using it and follow any adjacent format files in the same directory.

- `grill-with-docs`: challenge plans against repo language, update `CONTEXT.md`, and create ADRs only when decisions are hard to reverse, surprising without context, and based on a real trade-off.
